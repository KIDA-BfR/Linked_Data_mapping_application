# **Bioportal_ontology_recommender** Folder Structure

This directory contains the resources and implementation used for the ontology recommendation workflow based on the BioPortal API.

---

## 📁 Folders

### **Received files**
Contains outputs and term lists used during the ontology recommendation process:

- **`recommendations.json`** — Ontology recommendations generated by the BioPortal-based pipeline  
- **`terms_txt.zip`** — Complete list of terms used for mapping (openFSMR full dataset)
- **terms_txt.zip** - Content of `terms_txt.zip`

---

### **Uploaded files**
Includes input datasets required for running the ontology recommendation workflow:

- **`openFSMR_July_2025_groups_splitted.xlsx`** — Full openFSMR dataset in its raw, unprocessed format

---

## 📄 Files

### **Ontology_suggestions.ipynb**
Notebook implementing the ontology recommendation pipeline using the BioPortal API.  
It processes input terms, queries BioPortal, and generates ranked ontology suggestions.
