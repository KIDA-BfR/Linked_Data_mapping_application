# Folder Structure

## 📄 Files

### **Bioportal_communication_term_and_definition_search.ipynb**
This notebook provides code to:
1. Perform string-based term matching (e.g., label and synonym lookup) across a list of ontologies.
2. Retrieve the definition of a matched term, including resolving definitions through cross-ontology mappings when the original ontology does not provide one.

### **Bioportal_communication_term_search.ipynb**

This notebook implements string-based matching (e.g., label and synonym lookup) across a list of ontologies to identify potential matches for a given term.
